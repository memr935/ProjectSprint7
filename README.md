# Análisis de Datos de Vehículos – Aplicación Web Interactiva

## Descripción del proyecto

Este proyecto es una **aplicación web interactiva** desarrollada con **Streamlit**, cuyo objetivo es analizar un conjunto de datos de anuncios de venta de vehículos.

La aplicación permite explorar y visualizar información del conjunto de datos mediante **gráficos interactivos**, facilitando el análisis exploratorio de datos de forma visual e intuitiva.

---

## Funcionalidad que proporciona

La aplicación ofrece las siguientes funcionalidades:

- **Histogramas** para visualizar la distribución de variables numéricas, como el kilometraje de los vehículos.
- **Gráficos de dispersión** para identificar relaciones entre variables, por ejemplo, entre el precio y el kilometraje.
- **Casillas de verificación interactivas** que permiten al usuario mostrar u ocultar los gráficos.
- **Análisis exploratorio de datos** mediante visualizaciones dinámicas e interactivas.
